## Creating Your First Task

Now it's time to create your own task! Follow these steps to add a new task to your project.

### Step 1: Create the Directory

1. Navigate to your project directory (or create a new one)
2. Create a new directory following the naming convention:
   ```
   2024-01-05-0001_my-first-task/
   ```

### Step 2: Create README.md

Create a `README.md` file with the following structure:

```markdown
---
title: My First Task
status: Backlog
output: ''
assignee: Your Name
size: S
planned_start: '2024-01-05'
planned_end: '2024-01-06'
dependencies: []
pinned: false
---

## Task Description

Write a clear description of what this task involves.

### Acceptance Criteria

- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

### Notes

Add any additional notes, resources, or comments here.
```

### Step 3: View Your Task

1. Save the file
2. Refresh your Mica interface
3. Your new task should appear in the timeline!

### Exercise

Try creating a task for something you're currently working on. Experiment with different properties and see how they affect the visualization.

**Challenge**: Create a task with dependencies and see how the Gantt chart shows the relationships.
