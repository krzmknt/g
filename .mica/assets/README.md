# Assets Directory

This directory contains shared assets for your Mica project.

## Structure

- `images/` - Images, diagrams, screenshots
- `documents/` - PDF files, specifications, reports

## Image Guidelines

- **File Formats**: Use PNG for screenshots, SVG for diagrams, JPG for photos
- **File Naming**: Use descriptive, kebab-case names
- **Size**: Optimize images for web use
- **Alt Text**: Always provide meaningful alt text in markdown

## Sample Files

- `sample-architecture.svg` - Example system architecture diagram
- `sample-ui-mockup.svg` - Example user interface mockup

## Usage

Reference images in your task files using relative paths:

```markdown
![Architecture Diagram](../../assets/images/sample-architecture.svg)
```
