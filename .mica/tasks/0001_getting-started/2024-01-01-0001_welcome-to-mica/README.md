### What is Mica?

Mica is a markdown-based project management tool that combines the simplicity of Markdown with powerful Gantt chart visualization.

### Key Features

- **Markdown-based**: Write tasks and documentation in familiar Markdown syntax
- **Visual Timeline**: Automatic Gantt chart generation from your task data
- **Flexible Organization**: Organize tasks in categories and projects
- **Rich Content**: Support for images, tables, links, and more
- **Version Control Friendly**: Plain text files work great with Git

### Getting Started

1. **Explore the Examples**: Check out the sample projects in this directory
2. **Create Your First Task**: Copy an existing task and modify it
3. **Start the Server**: Run `mica start` to view your projects
4. **Learn the Syntax**: Review the documentation examples

Let's get started with your project management journey!
