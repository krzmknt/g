# Mica Project

Welcome to your Mica project! This directory contains your project management data.

## Getting Started

1. **Start the application**: Run `mica start` in your terminal
2. **Open your browser**: Navigate to http://localhost:3110
3. **Explore examples**: Check out the sample projects in the `examples/` directory

## Creating Your Own Projects

1. Create a new category directory under `.mica/tasks/` with format: `[category_id]_[category_name]/`
2. Add task directories following the pattern: `[task_id]_[task_name]/README.md`
3. Use the frontmatter format to define task properties
4. Add your content using Markdown
